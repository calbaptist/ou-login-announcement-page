Welcome to OU Campus!

OU Campus by OmniUpdate is the web content management system (CMS) used to manage the the calbaptist.edu website.

How do I get started?

In order to use this system, you must have be an authorized user. If you are an authorized user, you can login using your CBU username and password.

What if I need help?

If you have any questions or have any issues that come up while using the CMS, please contact Waylon Baumgardner, Senior Web Services Manager in CBU's Marketing department at 951-343-4876 or wbaumgardner@calbaptist.edu.

If you want to request a website update or other web related request, please submit that to the CBU Marketing Project Request System.
